# Fraud Pattern Detection & Automated Reporting Pipeline

## Overview
A Python and SQL based data pipeline that simulates payment transaction data, detects fraud signals, and automates weekly business review (WBR) style reports — mirroring real world fraud monitoring workflows used in fintech environments.

## Tech Stack
- Python
- PostgreSQL
- SQL
- pandas
- Plotly

## Features
- Synthetic payment transaction dataset generation
- SQL based velocity checks and outlier detection
- Merchant pattern analysis to surface anomalous behaviour
- Automated weekly summary report with trend charts
- Designed to scale for real world fraud and abuse prevention use cases
